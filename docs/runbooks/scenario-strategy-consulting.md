# 🏛️ Runbook: Strategy Consulting Engagement

> **Mode**: NEXUS-Consulting | **Duration**: 4-8 weeks | **Agents**: 10-15

---

## Scenario

You are delivering a strategic advisory engagement: market entry strategy, digital transformation roadmap, organizational restructuring, M&A due diligence, or competitive positioning. Output must provide actionable recommendations backed by rigorous analysis.

## Agent Roster

### Core Team (Always Active)
| Agent | Role |
|-------|------|
| Business Strategist | Lead strategist — competitive analysis, strategic frameworks |
| Market Researcher | Market sizing, trend analysis, competitor benchmarking |
| Financial Analyst | Financial modeling, valuation, business case |
| Executive Summary Generator | Board-ready summaries and presentations |

### Domain Specialist (Choose 2-4 Based on Engagement)
| Agent | Role |
|-------|------|
| Industry Analyst (sector-specific) | Industry dynamics, regulatory landscape |
| Technology Strategist / CTO Advisor | Technology assessment, digital maturity |
| Operations / Supply Chain Specialist | Operational efficiency, supply chain analysis |
| Organizational Design / HR | Org structure, talent strategy, change management |
| Legal & Compliance | Regulatory risk, compliance assessment |
| ESG / Sustainability | ESG strategy, sustainability roadmaps |
| Risk Management | Risk assessment, mitigation strategies |
| M&A / Corporate Finance | Deal structuring, synergy analysis |

### Quality & Delivery (Phase 4-5)
| Agent | Role |
|-------|------|
| Reality Checker | Stress-test assumptions — "what if the market grows at 2% not 8%?" |
| Presentation Designer | Slide deck design, data visualization, narrative flow |
| Project Manager | Timeline, client communication, deliverable tracking |

## Pipeline Phases

| Phase | Duration | Activities |
|-------|----------|------------|
| **0 - Discovery** | 1 week | Client intake, problem framing, hypothesis generation, data room review |
| **1 - Strategy** | 1 week | Analytical framework selection, work plan, team assignment, client alignment |
| **2 - Foundation** | 1 week | Data collection, market analysis, financial baseline, stakeholder interviews |
| **3 - Build** | 2-3 weeks | Core analysis, scenario modeling, option generation, recommendation development |
| **4 - Hardening** | 1 week | Stress-test assumptions, sensitivity analysis, fact-check, peer review, rehearse |
| **5 - Launch** | 2-3 days | Final presentation to client, board deck, written report delivery |
| **6 - Operate** | 2-4 weeks | Implementation support, follow-up Q&A, progress check-in |

## Quality Gates

| Gate | Check |
|------|------|
| 0→1 | Problem framed. Key questions defined. Client aligned on scope. Data accessed. |
| 1→2 | Framework selected and justified. Work plan with deadlines. All stakeholders identified. |
| 2→3 | Data verified. Financial baseline complete. No "we need more data" gaps. |
| 3→4 | All analysis complete. At least 2 options per decision. Recommendations drafted. |
| 4→5 | Assumptions stress-tested. Sensitivity analysis complete. Presentation rehearsed. |
| 5→6 | Client feedback incorporated. Implementation roadmap delivered. |

## Key Deliverable

- Board-level presentation deck (20-30 slides)
- Written report with analysis and appendices
- Financial model (if applicable)
- Implementation roadmap with milestones
- Executive summary (2 pages max)
