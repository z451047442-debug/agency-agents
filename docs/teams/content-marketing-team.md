# 内容营销团队

## 场景
全渠道内容营销战役：从内容策略到多渠道分发、SEO、社交媒体、邮件营销

## 团队成员 (8-12人)

| 角色 | Agent | 职责 |
|------|-------|------|
| 🎯 策略 | `marketing-content-strategist` | 内容策略、编辑日历、内容支柱规划 |
| ✍️ 文案 | `marketing-content-creator` | 博客、白皮书、案例研究、落地页文案 |
| 🔍 SEO | `marketing-seo-specialist` | 关键词策略、技术 SEO、内容优化 |
| 📧 邮件 | `marketing-email-marketing` | 邮件序列、Newsletter、A/B 测试 |
| 🐦 Twitter | `marketing-twitter-engager` | 实时互动、Thread 策略、行业对话 |
| 📸 Instagram | `marketing-instagram-curator` | 视觉内容、Stories、Reels |
| 🎵 TikTok | `marketing-tiktok-strategist` | 短视频内容、趋势跟踪 |
| 🎬 视频 | `creative-videographer` | 长视频、教程、客户证言视频 |
| 📊 分析 | `marketing-analytics` | 内容 ROI、渠道归因、受众洞察 |
| 📈 增长 | `marketing-growth-hacker` | 内容分发策略、SEO 增长、病毒传播 |
| 🎨 设计 | `design-visual-storyteller` | 信息图、社交媒体素材、视觉品牌 |
| 📡 社交 | `marketing-social-media-strategist` | 跨平台协调、发布日历、社区管理 |

## 工作流

```
Week 1: 内容策略 + 编辑日历 + 关键词研究
Week 2: 支柱内容创作 (博客/白皮书) + SEO 优化
Week 3: 社交媒体内容制作 + 邮件序列设计
Week 4: 多渠道分发 + 社区互动
Week 5: 数据分析 + ROI 报告 + 策略调整 (循环)
```

## 内容类型矩阵

| 内容类型 | 负责 Agent | 频率 | 分发渠道 |
|---------|-----------|------|---------|
| 深度博客 | Content Creator + SEO | 2x/周 | 官网 Blog |
| Newsletter | Email Marketer | 1x/周 | 邮件 |
| Twitter Thread | Twitter Engager | 3x/周 | Twitter/X |
| Instagram 视觉 | Instagram Curator + Visual Storyteller | 5x/周 | Instagram |
| TikTok 短视频 | TikTok Strategist + Video Producer | 3x/周 | TikTok |
| 案例研究 | Content Creator | 1x/月 | 官网 + LinkedIn |
| 白皮书/指南 | Content Creator + SEO | 1x/月 | 官网 (门控) |

## 决策点

| 决策 | 时机 | 决策人 |
|------|------|--------|
| 目标受众和内容支柱 | Week 1 | Content Strategist |
| 内容日历审批 | Week 1 | Studio Producer |
| 付费推广预算分配 | Week 1 | Growth Hacker + Finance Tracker |
| 渠道策略调整 | Week 5 | Growth Hacker + Analytics Reporter |

## 常见陷阱

| 陷阱 | 缓解 |
|------|------|
| 内容与 SEO 脱节 | 每篇内容创建前，先过 SEO 关键词清单 |
| 各渠道内容不一致 | 内容日历作为单一入口，所有 Agent 同步 |
| 只做内容不做分发 | "70/30 规则"：30% 创作时间，70% 分发推广 |
| 不追踪内容 ROI | 每个内容必须绑定 UTM + 转化目标 |
