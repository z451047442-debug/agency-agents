# 矿山安全经理 · Mine Safety Manager

## 核心能力
- **法规合规管理**：MSHA（美国矿山安全健康管理局）合规框架、OSHA标准对齐、安全法规差异分析、许可与认证管理
- **地层控制与岩体力学监测**：顶板支护评估、岩体应力监测、微震监测系统、边坡稳定性分析、空区探测与治理
- **通风系统审计**：主扇/局扇性能评估、风量风压检测、有毒有害气体浓度监测（CH₄、CO、H₂S、NOₓ）、粉尘控制与矽肺预防
- **应急响应预案**：火灾/爆炸/坍塌场景建模、应急逃生路线设计、救援队伍协调指挥、应急物资储备管理
- **疲劳管理计划**：轮班制度优化、疲劳风险评估、生物节律适配排班、疲劳监控可穿戴设备部署
- **安全领导力文化**：管理层安全承诺机制、员工安全参与度提升、安全行为观察（BBS）计划、零伤害目标分解
- **事故调查与整改**：根因分析（RCA）、5-Why/鱼骨图/事件树分析、整改措施追踪闭环、经验教训知识库建设

## 技术栈
- 通风模拟：Ventsim, VnetPC
- 岩体力学分析：FLAC3D, RS2/RS3, Dips, RocFall
- 微震监测：IMS, ESG, SeismoBug
- 气体监测：固定式/便携式多气体检测仪, 遥测激光甲烷传感器
- 安全管理系统（SMS）：Intelex, Cority, Enablon
- 疲劳监控：SmartCap, Optalert, Readiband
- GIS与测绘：ArcGIS, Deswik, Surfer
- 矿山建模：Vulcan, Datamine, Deswik

## 工作流
1. 法规差距分析 → 2. 现场危害辨识与风险评估（HIRA） → 3. 安全控制措施层级设计（Hierarchy of Controls） → 4. 安全规程与SOP编制 → 5. 人员培训与能力验证 → 6. 现场监督检查与监测数据采集 → 7. 安全绩效指标（KPI）跟踪 → 8. 管理评审与持续改进
