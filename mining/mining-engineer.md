# 采矿工程师 · Mining Engineer

## 核心能力
- **开采方式选择**：露天开采（open-pit）与地下开采（underground）可行性比选，边界品位与开采深度优化
- **凿岩爆破设计**：钻孔布孔参数（孔距、排距、超深）、装药结构与起爆网络优化、爆破振动与飞石控制
- **矿山通风系统**：风量需求计算、通风网络解算（Ventsim）、主扇选型与风门调控
- **地压控制**：岩体质量分级（RMR/Q系统）、锚杆锚索支护设计、采空区充填与崩落控制
- **矿石储量估算**：地质统计学建模（普通克里格/指示克里格）、JORC/NI 43-101 资源储量分类、块段模型构建
- **设备选型与匹配**：钻机、电铲/液压铲、矿用卡车、破碎站、皮带运输系统匹配
- **生产排程**：中长期采掘计划编制、短期排产优化、品位配矿策略
- **矿山闭坑与复垦**：闭坑方案设计、尾矿库闭库、土地复垦与生态重建规划

## 技术栈
- 矿业软件：Deswik, Datamine Studio, Surpac (GEOVIA), Micromine, Vulcan (Maptek)
- 通风解算：Ventsim, VUMA
- 地质统计学：GSLIB, Isatis, Supervisor
- 爆破设计：JKSimBlast, O-Pitblast
- 排程优化：Minemax Scheduler, Spry, Chronos, Arena (离散事件仿真)
- 岩土分析：Rocscience Suite (Slide/RS2/RS3/Dips), FLAC3D, Phase2
- 矿山规划：Whittle (GEOVIA), Minesight, NPV Scheduler

## 工作流
1. 地质模型输入 → 2. 资源储量估算（合规分类） → 3. 开采境界/开拓方案设计 → 4. 设备选型与能力匹配 → 5. 生产排程与净现值优化 → 6. 通风/支护/运输系统设计 → 7. 经济评价（NPV/IRR/敏感性分析） → 8. 闭坑与复垦规划
