# 📚 Runbook: Course / Curriculum Design

> **Mode**: NEXUS-Education | **Duration**: 3-6 weeks | **Agents**: 8-12

---

## Scenario

You are designing an educational course, training program, or full curriculum. Applies to: university courses, corporate training, online courses (MOOC), K-12 unit plans, professional certification prep, vocational training modules.

## Agent Roster

### Core Team (Always Active)
| Agent | Role |
|-------|------|
| Curriculum Designer | Learning objectives, course structure, assessment design |
| Subject Matter Expert (discipline-specific) | Content accuracy, depth, relevance |
| Education Technology (EdTech) | LMS setup, interactive content, multimedia integration |
| Online Learning Designer | Learning experience design, engagement strategies |

### Teaching Specialist (Choose 1-2 Based on Level)
| Agent | Role |
|-------|------|
| K-12 Subject Teaching | Primary/secondary pedagogy, classroom management |
| Higher Education Teaching | University-level design, seminar facilitation |
| Language Teaching | Language acquisition methodology |
| STEM & Engineering Education | Lab design, project-based learning |
| Medical Education | Clinical teaching, OSCE design |
| Vocational Education | Skills-based training, competency assessment |

### Quality & Delivery (Phase 4-5)
| Agent | Role |
|-------|------|
| Reality Checker | Learning objective alignment — does every assessment measure a stated objective? |
| Assessment Specialist | Test validity, rubric design, item analysis |
| Special Education Specialist | Accessibility, Universal Design for Learning (UDL) |
| Content Reviewer (domain) | Peer review of content accuracy |

## Pipeline Phases

| Phase | Duration | Activities |
|-------|----------|------------|
| **0 - Discovery** | 2-3 days | Learner needs analysis, prerequisite assessment, stakeholder requirements, existing curriculum audit |
| **1 - Strategy** | 2-3 days | Learning objectives (SMART), competency framework, course map, assessment strategy, modality |
| **2 - Foundation** | 3-5 days | Unit outlines, content sourcing, LMS setup, template design, assessment blueprint |
| **3 - Build** | 1-3 weeks | Content creation (lectures, readings, activities, assignments), multimedia, quiz authoring |
| **4 - Hardening** | 3-5 days | Accessibility review (UDL), content accuracy, alignment audit (objectives→assessments→activities) |
| **5 - Launch** | 1-2 days | Course published to LMS, learner onboarding, instructor guide released |
| **6 - Operate** | Ongoing | Learner feedback, assessment performance analysis, content iteration, discussion facilitation |

## Quality Gates

| Gate | Check |
|------|------|
| 0→1 | Learner personas defined. Prerequisites clear. Stakeholder requirements documented. |
| 1→2 | Objectives are SMART. Assessment strategy aligns to objectives. Modality decided. |
| 2→3 | Unit outlines complete. Assessment blueprints exist for every assessment. LMS configured. |
| 3→4 | All content created. All assessments have rubrics. Multimedia functional. No placeholder text. |
| 4→5 | UDL checklist passed. Content accuracy verified. 100% alignment: assessments measure stated objectives. |
| 5→6 | Course accessible to learners. Instructor guide complete. Onboarding materials live. |

## Key Deliverable

- Complete course in LMS with:
  - Syllabus / course overview
  - Unit/Module learning materials (lectures, readings, activities)
  - Assessments with rubrics and answer keys
  - Instructor guide / facilitation notes
  - Accessibility compliance report
