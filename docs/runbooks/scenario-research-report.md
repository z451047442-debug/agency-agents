# 📄 Runbook: Research Report / White Paper

> **Mode**: NEXUS-Research | **Duration**: 2-4 weeks | **Agents**: 8-12

---

## Scenario

You need a comprehensive research report, white paper, or industry analysis. Output must be academically rigorous, well-structured, and ready for publication. Applies to: market research, technical white papers, policy analysis, academic literature reviews, competitive intelligence reports.

## Agent Roster

### Core Team (Always Active)
| Agent | Role |
|-------|------|
| Academic Research Scientist | Research methodology, literature review, data analysis |
| Market Researcher | Market intelligence, competitor analysis, trend identification |
| Technical Writer / Documentation Engineer | Report structure, clear writing, citations |
| Executive Summary Generator | C-suite summary, key findings synthesis |

### Domain Expert (Choose 1-3 Based on Topic)
| Agent | Role |
|-------|------|
| Financial Analyst | Finance/economics research |
| Healthcare / Medical Specialist | Medical/clinical research |
| Legal / Compliance | Regulatory/policy research |
| Data Scientist / Statistician | Statistical analysis, data visualization |
| Engineering Specialist | Technical/engineering white papers |
| ESG & Sustainability | Sustainability/environmental research |
| Military / Defense Strategist | Defense/security research |
| Education Specialist | Educational research |

### Quality & Review (Phase 4)
| Agent | Role |
|-------|------|
| Reality Checker | Evidence verification — every claim needs a source |
| Peer Reviewer (domain) | Domain accuracy review |
| Editor / Proofreader | Grammar, citation format, structural coherence |

## Pipeline Phases

| Phase | Duration | Activities |
|-------|----------|------------|
| **0 - Discovery** | 2-3 days | Define research question, scope, methodology. Identify data sources and key literature. |
| **1 - Strategy** | 1-2 days | Outline structure, assign sections, define quality criteria, set citation format. |
| **2 - Foundation** | 2-3 days | Literature collection, data gathering, tool setup (Zotero, statistical tools). |
| **3 - Build** | 1-2 weeks | Write sections, analyze data, create visualizations, draft conclusions. |
| **4 - Hardening** | 2-3 days | Fact-check all claims, verify citations, proofread, format consistency, peer review. |
| **5 - Launch** | 1 day | Final PDF/HTML output, executive summary, distribution to stakeholders. |
| **6 - Operate** | Ongoing | Track citations, update with new data, respond to feedback. |

## Quality Gates

| Gate | Check |
|------|-------|
| 0→1 | Research question is specific and answerable. Primary sources identified. |
| 1→2 | Outline complete. Each section has an owner. Data sources verified as accessible. |
| 2→3 | All sources collected and catalogued. Data clean and ready for analysis. |
| 3→4 | All sections drafted. Every claim has citation. No "TBD" sections. |
| 4→5 | 100% citations verified. Grammar and format pass. Peer review complete. Summary accurate. |
| 5→6 | Report published. Readers can access. No broken links. |

## Key Deliverable

- Executive summary (1-2 pages)
- Main body with clear section structure
- Data visualizations (charts, tables)
- Complete bibliography/citations
- Appendices (methodology details, raw data)
